# Documentação do Projeto Lago Verde

Este diretório centraliza os principais documentos de apoio à decisão, arquitetura e execução do projeto.

## Estrutura

- [ADR](./adr/README.md)
  Registros formais de decisões arquiteturais.
- [Levantamento de Tecnologias](./discovery/levantamento-tecnologias.md)
  Análise comparativa e recomendação da stack.
- [Visão Geral da Solução](./architecture/visao-geral-da-solucao.md)
  Descrição da arquitetura proposta para o sistema.
- [Desenho Técnico de Ambientes e Deploy](./architecture/desenho-tecnico-ambientes-e-deploy.md)
  Detalhamento operacional dos ambientes, publicação, deploy e rollback.
- [Plano de Execução](./process/plano-de-execucao.md)
  Fluxo de trabalho sugerido para design, desenvolvimento, validação e entrega.
- [Perguntas para Reunião de Acesso Operacional](./process/perguntas-reuniao-acesso-operacional.md)
  Pauta objetiva para alinhar permissões, scripts, deploy, logs, backup e responsabilidades.

## Observação

Os documentos se complementam:

- o `levantamento de tecnologias` avalia opções;
- o `ADR` registra a decisão tomada;
- a `visão geral da solução` descreve como a arquitetura será montada;
- o `plano de execução` descreve a ordem prática de trabalho do time.
